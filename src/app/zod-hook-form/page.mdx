# Zod + React Hook Form example

Form input handling with [react-hook-form](https://react-hook-form.com/) and [Zod](https://zod.dev/) validation with [vovk-zod](https://www.npmjs.com/package/vovk-zod). The form is validated using RPC method schema emitted by **vovk-zod** with [Ajv](https://ajv.js.org/), [Ajv formats](https://www.npmjs.com/package/ajv-formats) and [Ajv resolver](https://www.npmjs.com/package/@hookform/resolvers#ajv). When the form is submitted, the request input is validated with the generic `validateOnClient` and, when the request is sent to the server, it is validated again with the original Zod schema.
