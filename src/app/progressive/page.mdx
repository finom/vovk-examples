# Progressive Response example

Demonstrates an experimental feature called "Progressive Response" that implemented with `JSONLinesResponse` on the server and handled with `progressive` function on the client. For demonstational purposes users and tasks resolve in random order, using `setTimeout` with a random delay of 0-10 seconds. Reload the page to see the effect again.
