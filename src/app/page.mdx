# Vovk.ts examples

Random examples and proofs of concepts. The RPC library is published as [vovk-examples](https://npmjs.com/package/vovk-examples) package to be used for live demos on [vovk.dev](https://vovk.dev). All code blocks are pulled from [the repository](http://github.com/finom/vovk-examples) and represent the latest state of the code.

**Other repos:**

- [vovk-hello-world](https://github.com/finom/vovk-hello-world) - an example of a "Hello World" application with most features enabled.
- [vovk-multitenant-example](https://github.com/finom/vovk-multitenant-example) - an example of a multi-tenant application.
