# Vovk.ts examples

Random and uncategorized examples of Vovk.ts usage for concept proofs and copy-pastable code that is used on [vovk.dev](https://vovk.dev). The examples are published as [vovk-examples](https://npmjs.com/package/vovk-examples) package to be used for live demos. All code examples are pulled from [the repository](http://github.com/finom/vovk-examples).
