# Vovk.ts Examples

Random examples and proofs of concepts. The RPC library is published as [vovk-examples](https://npmjs.com/package/vovk-examples) package to be used for live demos on [vovk.dev](https://vovk.dev). All code blocks are pulled from [the repository](http://github.com/finom/vovk-examples) and represent the latest state of the code.

## Other Examples

- [vovk-hello-world](https://github.com/finom/vovk-hello-world) - an example of a "Hello World" application with most features enabled. [Read more](https://vovk.dev/hello-world).
- [vovk-multitenant-example](https://github.com/finom/vovk-multitenant-example) - an example of a multi-tenant application. [Read more](https://vovk.dev/multitenant).
- [vovk-kanban-demo](https://github.com/finom/vovk-kanban-demo) - a Kanban board application with AI features. [Read more](https://vovk.dev/realtime-ui).
